# Prasad Raju G — Portfolio

Personal portfolio website built with [al-folio](https://github.com/alshedivat/al-folio), a Jekyll theme for academics.

**Live Site:** [https://your-domain.github.io](https://your-domain.github.io)

## Local Development

```bash
docker compose pull && docker compose up
```

Site runs at `http://localhost:8080`.

## Tech Stack

- **Framework:** Jekyll
- **Theme:** al-folio
- **Hosting:** GitHub Pages

## Project Structure

| Directory | Description |
|-----------|-------------|
| `_pages/` | Static pages (about, CV, etc.) |
| `_posts/` | Blog posts |
| `_projects/` | Project entries |
| `_bibliography/` | BibTeX publications |
| `_data/` | YAML data files |
| `_includes/` | Liquid template partials |
| `_layouts/` | Page layouts |
| `assets/` | Images, JSON, CSS |
