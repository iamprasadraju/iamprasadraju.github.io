# Phonebook

A comprehensive contact management solution built with Flask.

## Features

This application provides a complete phonebook management system with modern web technologies.

### Core Functionality

- **User Authentication**: Secure login and registration system
- **Contact Management**: Create, read, update, and delete contacts seamlessly
- **Password Reset**: Easy password recovery with expiration-based security
- **Real-time Validation**: Instant feedback on user inputs

### Technical Stack

- Flask web framework
- SQL database for data persistence
- Responsive design principles
- QR code integration for quick contact sharing
