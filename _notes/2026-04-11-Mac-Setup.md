Step-by-step instructions to setup new Mac

### CLI Tools
---

1. [Xcode Command Line Tools](https://mac.install.guide/commandlinetools/) ``` 
    xcode-select –-install
  ``` it comes with `clang`, `gcc`, `clang++`, `g++`, `git` and more ...

    **List of Xcode Command Line Tools**
    
    ```bash
      ls /Library/Developer/CommandLineTools/usr/bin/
    ```

2. [Homebrew](https://en.wikipedia.org/wiki/Homebrew_(package_manager)) is a free and open-source software package management system that simplifies the installation of software on Apple's operating system, macOS, as well as Linux. [official Homebrew site](https://brew.sh/)
    
3. [Git](https://git-scm.com/) Already comes with Xcode CLT

4. [Gen CLI](https://github.com/iamprasadraju/gen-cli) (My own CLI tool)```pip install gen-cli```

5. [opencode](http://opencode.ai/) is an open source agent that helps you write code in your terminal, IDE, or desktop.
### Softwares and Applications
---

1. [Brave](https://brave.com/) A Browser.
2. [OBS](https://obsproject.com/) Free and open source software for video recording and live streaming.
3. [Zed](https://zed.dev/) is a Rust based code editor.
4. [Discord](https://discord.com/) is a free voice and text communication application and digital distribution platform.
5. [Spotify](https://open.spotify.com/download) is a popular digital audio streaming service for music listening.
6. [Obsidian](https://obsidian.md/) stores notes privately on your device as markdown files.
7. [Battery Toolkit](https://github.com/mhaeuser/Battery-Toolkit) Control the platform power state of your Apple Silicon Mac. Battery Toolkit allows specifying a hard limit past which battery charging will be turned off.
8. [MacDroid](https://apps.apple.com/us/app/macdroid-manager-for-android/id1476545828?mt=12) the ultimate solution for transferring files between Android and Mac.
9. [Rectangle](https://rectangleapp.com/) Move and resize windows in macOS using keyboard shortcuts or snap areas.
10. [Mos](http://mos.caldis.me/) is a free, open-source macOS utility that makes wheel scrolling feel closer to a trackpad, without getting in your way.


### Programming Languages
---

1. [Python](https://www.python.org/) ```brew install python```
2. [Ruby](https://www.ruby-lang.org/en/documentation/installation/#homebrew) ```brew install ruby```
3. [C](https://en.wikipedia.org/wiki/C_(programming_language)) Already comes With Xcode CLT i.e `gcc` and `clang` 
4. [C++](https://en.wikipedia.org/wiki/C%2B%2B) Already comes With Xcode CLT i.e `g++` and `clang++` 

### Libraries
---

1. [Numpy](https://numpy.org/doc/stable/) the fundamental package for scientific computing with Python.```pip install numpy```
2. [Pandas](https://pandas.pydata.org/docs/) is a fast, powerful, flexible and easy to use open source data analysis and manipulation tool,
built on top of the Python programming language. ```pip install pandas```
3. [Scikit-learn](https://scikit-learn.org/stable/user_guide.html) is an open source machine learning library that supports supervised and unsupervised learning. It also provides various tools for model fitting, data preprocessing, model selection, model evaluation, and many other utilities. ```pip install scikit-learn```
4. [Seaborn](https://seaborn.pydata.org/index.html) is a Python data visualization library based on matplotlib. It provides a high-level interface for drawing attractive and informative statistical graphics. ```pip install seaborn```
5. [Matplotlib](https://matplotlib.org/) is a comprehensive library for creating static, animated, and interactive visualizations in Python. ```pip install matplotlib```
6. [Pytorch](https://docs.pytorch.org/docs/stable/index.html#) is an open-source optimized tensor library for deep learning using GPUs and CPUs **Deep learning library** ```pip install torch```
7. [Tensorflow](https://www.tensorflow.org/api_docs) is an open-source, end-to-end machine learning platform developed by Google, specializing in deep neural networks, numerical computation, and large-scale machine learning. ```pip install tensorflow```
8. [Scipy](https://docs.scipy.org/doc/scipy/) is a collection of mathematical algorithms and convenience functions built on NumPy . It adds significant power to Python. ```pip install scipy```
9. [Jupyter Notebook](http://jupyter.org/) is the latest web-based interactive development environment for notebooks, code, and data. Its flexible interface allows users to configure and arrange workflows in data science, scientific computing, computational journalism, and machine learning. A modular design invites extensions to expand and enrich functionality. ```pip install notebook```
10. [OpenCV](https://docs.opencv.org/4.x/) is a library of programming functions mainly for real-time [computer vision](https://en.wikipedia.org/wiki/Computer_vision)  ```pip install opencv-python```
