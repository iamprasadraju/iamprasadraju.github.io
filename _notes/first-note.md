Welcome to your notes section. This is a simple note written in Markdown.

Notes work just like blog posts but are organized separately under the `/notes/` URL. Use them for quick write-ups, personal reminders, or lightweight documentation.

## How to create a new note

Create a new file in the `_notes/` directory with this front matter:

```yaml
---
layout: post
title: your note title
date: YYYY-MM-DD
description: a short description
tags: [tag1, tag2]
categories: [notes]
---
```

Then write your content in Markdown below the front matter.
