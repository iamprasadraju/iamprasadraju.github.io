Phonebook

The Flask-based Phonebook Web App enables users to efficiently manage contacts with full CRUD functionality, user authentication, password reset, and real-time input validation for a seamless experience.

  • User Authentication: Secure user login and registration.
  • Contact Management: Create, read, update, and delete contacts.
  • Password Reset: Easily reset passwords with expiration for security.
  • Real-time Validation: Validate user inputs in real-time with instant feedback.

Music Genre Classification Using Convolution Neural Networks

Music genre classification is the task of automatically identifying the genre of a song based on its audio features. Convolutional Neural Networks (CNNs) are well-suited for this task because they can learn hierarchical features from spectrogramsâ€”visual representations of audio signals.

  • Spectrogram-Based Learning: Converts audio signals into spectrogram images, allowing CNNs to extract frequency and temporal patterns.
  • Deep Feature Extraction: Uses convolutional layers to automatically learn genre-specific characteristics from spectrograms without manual feature engineering.
  • Automated Classification: Trains on labeled datasets (e.g., GTZAN, FMA) and predicts the genre of new audio inputs with high accuracy.
